# 婚礼日记

![](assets/2025-11-03-15-32-44.png){width=200}

<figurecaption>💐🤲</figurecaption>
</figure>

婚礼定在十月的第三天，大概是个黄道吉日。可天气不甚好，晚间还下了雨，不知道这吉日是怎么算的。婚礼本该只有一天，可我被我哥抓去当了伴郎，无奈跟着连忙了三天。打扫卫生，布置婚房，跑老远试了伴郎的衣服，还去先人坟前祭拜了一下。不过这些都还好，婚礼当天才是最手忙脚乱的。

七点过就要起床，这还算上我们不用化妆已经偷懒多睡了一会儿，不知道新娘几点就起了。大约八点，跟拍的摄影师加入了战场。非常熟练地拍了一系列分镜，俗套的像个短剧。

摄影大哥按照预设的剧本拍了好一会儿，剧情终于发展到了高潮：出发接亲！收拾妥当，新郎就捧起花出发了。浩荡的车队打着双闪，还没转几个街角就停了下来。是的，新娘家不过一里开外。伴着浓浓的硝烟升起，新郎闪亮登场。
