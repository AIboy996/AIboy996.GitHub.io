# 装机必备
本文记录各个平台我用过的好用的软件。

如果您也有什么**宝藏软件**想要分享，请务必留下评论！！

## 全平台
> 主要是说Windows+MacOS，部分支持Linux及移动端

### [Apple Music](https://music.apple.com/)
唯一指定听歌软件，谁用谁说好。另外虽然是Apple的产品，但它是全平台可用的
### Clash
懂得都懂，clash是开源内核，在各个平台都有对应的客户端
### [LICEcap](https://www.cockos.com/licecap/)
超好用的GIF录屏软件
### [LocalSend](https://localsend.org)
局域网文件贡献，唯一真神。
### [Google Chrome](https://www.google.com/chrome/)
唯一指定的浏览器
### [Visual Studio Code](https://code.visualstudio.com/)
唯一指定的文本（代码）编辑器，插件丰富
### [MindLine](https://www.mindline.cn/)
平平无奇的思维导图软件，我脑抽了买了永久会员（当然也不是很贵），有云同步
### [Notion](https://www.notion.so/)
最好用的协作笔记
### [OneNote](https://www.onenote.com/)
微软的笔记软件，真全平台可用的云笔记
### [Plex](https://www.plex.tv/)
美观易用的个人媒体服务器，支持全平台客户端和服务端
### [QQ](https://im.qq.com/)
QQ-轻松做自己。意义不明的slogan
### [Qv2ray](https://github.com/Qv2ray/Qv2ray)
同上
### [Open Broadcaster Software](https://obsproject.com/)
唯一指定录屏软件
### [Snipaste](https://www.snipaste.com/)
截图软件，beta版本也已经支持MacOS了
### [Telegram](https://telegram.org/)
a new era of messaging
### [Iriun Webcam](https://iriun.com/)
局域网摄像头共享，可以把手机(iOS & Android)的摄像头共享给电脑使用
### [Termius](https://termius.com/)
ssh、sftp客户端
### [Typora](https://typora.io/)
值得掏腰包的markdown编辑软件
### [WeChat](https://weixin.qq.com/)
微信，是一个生活方式。以及唯一指定截图软件（
### [Zotero](https://www.zotero.org/)
文献管理工具
### [calibre](https://calibre-ebook.com/)
电子书库管理
### [git](https://git-scm.com/)
版本管理工具
### [iCloud](https://www.icloud.com/)
死贵，但是没法回头的云盘（Windows上也勉强能用，但是一坨）
### [massCode](https://masscode.io/)
代码片段管理
### [qBittorrent](https://www.qbittorrent.org/)
功能强大的种子下载客户端
### [坚果云](https://www.jianguoyun.com/)
非付费用户，配合Zotero用的，是个还不错的云盘
## Windows
### [7-zip](https://www.7-zip.org/)
压缩、解压缩
### [AquaSnap](https://www.nurgo-software.com/products/aquasnap)
窗口管理
### [Cygwin](https://www.cygwin.com/)
A large collection of GNU and Open Source tools which provide functionality similar to a Linux distribution on Windows.
### [DiskGenius](https://www.diskgenius.com/)
数据恢复、磁盘管理
### [Everything](https://www.voidtools.com/zh-cn/)
Win上的装机必备搜索软件，速度非常快！
### [Geek Uninstaller](https://geekuninstaller.com/)
好用的卸载工具
### [Fiddler](https://www.telerik.com/fiddler)
强大的网络代理工具
### [Lyricify](https://github.com/WXRIW/Lyricify-App)
可以显示iTunes的歌词，还可以检索网易云等平台等翻译版本
> 最新版本的Apple music已经可以自己显示歌词了！当然，还是Lyricify提供的网易云歌词更好～
### [PDFsam](https://pdfsam.org/)
pdf重组、编辑、拆分
### [Blueberry Airplay](https://deeprd.com/)
在Windows上接受AirPlay投屏信号
### [Potplayer](https://potplayer.daum.net/)
强大的媒体播放器
### [ReNamer](https://www.den4b.com/products/renamer)
巨好用的批量重命名工具，懒得写脚本的时候可以用一下
### [ScreenToGif](https://www.screentogif.com/)
超好用的GIF录屏软件、GIF编辑工具。可惜不支持MacOS。
### [SumatraPDF](https://www.sumatrapdfreader.org/free-pdf-reader)
唯一指定阅读器，轻便快捷。支持PDF, eBook (epub, mobi), comic book (cbz/cbr), DjVu, XPS, CHM以及常见的图片格式。
### [TrafficMonitor](https://github.com/zhongyang219/TrafficMonitor)
任务栏看看网速啥的，小巧玲珑
### [Wireshark](https://www.wireshark.org/)
强大的网络分析工具
### [WoMic](https://wolicheng.com/womic/)
局域网麦克风共享，可以把手机(iOS & Android)的麦克风共享给电脑使用
