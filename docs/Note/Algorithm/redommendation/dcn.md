# 特征交叉2：DCN
>
> 深度交叉网络

此前的召回模型、排序模型中有非常多的神经网络组件（例如双塔模型的两个塔），实际上可以使用任意类型的神经网络。最简单的就是全连接，在推荐系统中使用本节介绍的DCN效果要好得多。
