# 运筹学
> MANA130005，运筹学，周蓉

和统计沾边，但本身是其他学科（优化方向！）的基础课。

## 主要内容

- 线性规划
- 目标规划
- 整数规划
- 非线性规划
- 动态规划
- 图和网络分析
- 排队论
- 对策论

## 参考书
胡运权《运筹学教程》，没看过其他的课本，这本总体上感觉也还不错。

## 体会
主要内容就是**各类优化**，都是比较基础的内容。

- 线性规划手搓单纯形表还是很枯燥的
- 整数规划是个很难的话题，但是我们讲的不深入
- 动态规划是个好算法
- 图和网络算法很实用，最大流、最短路都是非常容易接受的重要算法！
- 排队论部分由于缺少随机过程的前置知识基本就是记住结论。听闻后续还有随机运筹学，大概才是正菜。
- 对策论讲的太简单
