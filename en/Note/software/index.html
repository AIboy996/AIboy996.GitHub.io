装机必备¶

本文记录各个平台我用的软件，不定期更新。

如果您也有什么宝藏软件想要分享，请务必留下评论！！

全平台¶

主要是说Windows+MacOS，部分支持Linux及移动端

Apple Music¶

唯一指定听歌软件，界面美观、简单易用。

虽然是Apple的产品，但它是全平台可用的

LICEcap¶

超好用的GIF录屏软件

LocalSend¶

局域网文件共享的唯一真神

Google Chrome¶

唯一指定的浏览器

Visual Studio Code¶

唯一指定的文本（代码）编辑器，插件丰富

Docker¶

虚拟容器

MindLine¶

平平无奇的思维导图软件，我脑抽了买了永久会员（当然也不是很贵），有云同步

Notion¶

最好用的协作笔记

OneNote¶

微软的笔记软件，真全平台可用的云笔记

Plex¶

最近Plex网络的问题有点劝退我，已转战Jellyfin

美观易用的个人媒体服务器，支持全平台客户端和服务端

Jellyfin¶

开源的个人媒体服务器（Emby的复刻版本），社区活跃、功能强大、开箱即用

QQ¶

QQ-轻松做自己。意义不明的slogan

Open Broadcaster Software¶

唯一指定录屏软件

Snipaste¶

截图软件，beta版本也已经支持MacOS了

Telegram¶

a new era of messaging

Iriun Webcam¶

局域网摄像头共享，可以把手机(iOS & Android)的摄像头共享给电脑使用

Termius¶

ssh、sftp客户端

Typora¶

值得为之掏腰包的markdown编辑软件

WeChat¶

微信，是一个生活方式。以及唯一指定截图软件（

Zotero¶

文献管理工具，搭配坚果云实现云同步

calibre¶

电子书库管理，搭配iCloud实现云同步

git¶

版本管理工具

iCloud¶

死贵，但是没法回头的云盘（Windows上也勉强能用，但是一坨）

massCode¶

代码片段管理，搭配iCloud实现云同步

qBittorrent¶

功能强大的种子下载客户端

坚果云¶

非付费用户，配合Zotero用的，是个还不错的云盘，主打办公

Alist¶

云盘集成，webdav映射

CasaOS¶

简洁易用的NAS中控，集成Docker功能

Clash¶

懂得都懂，clash是开源内核，在各个平台都有对应的客户端

Qv2ray¶

同上

Pocket¶

稍后再读

Windows¶

Chocolatey¶

Windows包管理工具，很多的开源软件都会提供choco的安装方式

7-zip¶

压缩、解压缩

AquaSnap¶

窗口管理

Cygwin¶

A large collection of GNU and Open Source tools which provide functionality similar to a Linux distribution on Windows.

DiskGenius¶

数据恢复、磁盘管理

Everything¶

Win上的装机必备搜索软件，速度非常快！

Geek Uninstaller¶

好用的卸载工具

Fiddler¶

强大的网络代理工具

Lyricify¶

可以显示iTunes的歌词，还可以检索网易云等平台等翻译版本

最新版本的Apple music已经可以自己显示歌词了！当然，还是Lyricify提供的网易云歌词更好～

PDFsam¶

pdf重组、编辑、拆分

Blueberry Airplay¶

在Windows上接受AirPlay投屏信号

Potplayer¶

强大的媒体播放器

ReNamer¶

巨好用的批量重命名工具，懒得写脚本的时候可以用一下

ScreenToGif¶

超好用的GIF录屏软件、GIF编辑工具。可惜不支持MacOS。

SumatraPDF¶

唯一指定阅读器，轻便快捷。支持PDF, eBook (epub, mobi), comic book (cbz/cbr), DjVu, XPS, CHM以及常见的图片格式。

TrafficMonitor¶

任务栏看看网速啥的，小巧玲珑

Wireshark¶

强大的网络分析工具

WoMic¶

局域网麦克风共享，可以把手机(iOS & Android)的麦克风共享给电脑使用

NSSM¶

NSSM - the Non-Sucking Service Manager，可以在Windows上方便地把.exe程序注册为系统服务（比Windows自带的好用太多

MiKTeX¶

TeX for Windows

MacOS¶

Homebrew¶

MacOS包管理工具，很多的开源软件都会提供brew的安装方式

AppCleaner¶

Mac上好用的软件卸载工具

Hammerspoon¶

MacOS自动化工具

IINA¶

强大的媒体播放器

Itsycal¶

任务栏日历插件，美观、实用

KeyCastr¶

键盘敲击可视化

Maccy¶

剪切板增强

OrbStack¶

Fast, light, simple Docker & Linux on macOS

Rectangle¶

窗口管理

Unarchiver¶

压缩、解压缩

iTerm2¶

终端软件

sketchybar¶

菜单栏软件，替代原生的菜单栏

MacTeX¶

TeX for Mac

iOS/iPadOS¶

部分支持MacOS

Bark¶

Webhook通知软件

GoodNotes¶

虽然GoodNotes 6狠狠恶心了用户一波，但不妨碍这是一个好的手写笔记软件

Yealico¶

建议搭配别人分享的规则使用

网站内容提取，信息聚合

Vidhub¶

媒体播放器，可以访问Jellyfin和Plex、Emby，还可以直接访问各种云盘文件

Infuse¶

功能同上，但是更贵

APTV¶

IP TV播放器，收看电视直播

Microsoft Remote Desktop¶

现在改名字了：Windows App Mobile

远程连接Windows

MNGA¶

NGA 论坛 iOS 开源客户端

Pendo¶

可爱的日记、代办软件

QQ邮箱¶

腾讯为数不多的良心软件（

Bangumi¶

追番进度管理

NetNewsWire¶

RSS阅读器

Pillow¶

睡眠记录分析

Microsoft Authenticator¶

保存的密码还支持浏览器插件自动填充，但是不支持2FA的自动填充

2FA、密码管理，支持云端备份

Shadowrocket¶

懂得都懂

Delta¶

GBA模拟器

XP3Player¶

可以用来玩krkr2引擎的galgame（例如魔法使之夜）

Picsew¶

长截图

Eye Measure¶

瞳距测量（AppStore真是什么鬼东西都有啊

Stream¶

抓包工具，简单易用~

Sonic Tools¶

这是一款用于声音、振动和磁场的分析仪。

MQTTool¶

MQTT消息队列小工具

ProDex¶

宝可梦图鉴

MusicMatch¶

听AppleMusic的时候实时翻译外文歌词

熊猫吃短信¶

短信过滤

Android¶

现在安卓用的比较少了😪

Tachiyomi¶

看漫画的软件，可以聚合各个漫画平台

Framelapse 2¶

延时摄影

Linux¶

一些终端工具，大部分都支持全平台

tmux¶

终端分屏

vim¶

文本编辑器

zoxide¶

cd的上位替代

neofetch¶

获取操作系统信息

zsh¶

Mac的默认Shell

powerlevel10k¶

zsh主题

fish¶

user-friendly Shell

miniconda¶

虚拟环境管理器，无root权限也可用！

wakeonlan¶

Windows上有一个替代

一个WOL的Perl脚本

ffmpeg¶

视频流处理

pandoc¶

文件格式转换

marp¶

Markdown Presentation，可以用Markdown文件快速制作演示文稿
